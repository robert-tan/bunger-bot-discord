## How to run
1. Clone this repo
2. Create a `.env` file in this directory and add the bot token as follows:
```
DISCORD_TOKEN=YOUR-TOKEN-HERE
```
3. Install dependencies (below)
4. Run `python3 bot.py`

## Dependencies
Python 3.8 (dev)
Discord.py
python-dotenv
